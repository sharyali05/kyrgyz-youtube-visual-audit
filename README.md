# Visual Semiotics and Language Ideologies in Kyrgyz Digital Media
### A Computational Audit of YouTube Children's Content

---

## Overview

This repository contains the data, code, and analysis for a cultural analytics study examining whether YouTube children's content recommended to Kyrgyz users differs systematically in its **visual aesthetics** across language groups — specifically comparing Kyrgyz-language, Russian-language, and other content.

This project extends prior algorithmic audit work (*Empire Amplifier*) which demonstrated that YouTube disproportionately recommends Russian-language content to Kyrgyz children even when they signal preferences for Kyrgyz-language content. Where that work focused on *which* content gets recommended, this project asks: **what does that content look like**, and what do those visual differences signal ideologically?

Ethnographic interviews and participant observation in Bishkek, Kyrgyzstan suggest that:
- Kyrgyz-language children's content (e.g., *Keremet Koch*) is perceived as "static," low-production-value, and associated with Soviet-era educational media
- Russian-language content is perceived as more dynamic, colorful, and professionally produced
- These aesthetic perceptions reinforce broader language ideologies in which Russian is associated with modernity and prestige, while Kyrgyz is increasingly domain-restricted to elder speech and traditional contexts

This study operationalizes those perceptions as measurable visual features extracted from **YouTube thumbnails** and tests whether they hold up computationally across a labeled corpus of YouTube videos. Thumbnails are used instead of full video frames because they are lightweight, fast to retrieve, and designed by creators to be visually representative of their content — making them valid proxies for color, brightness, and vlogger aesthetic features.

---

## Research Questions

1. **Color saturation**: Does Russian/international content show higher average color saturation than Kyrgyz-produced content?
2. **Luminance**: Does Russian/international content show higher average brightness than Kyrgyz-produced content?
3. **Vlogger aesthetics**: Do Russian-language videos more frequently deploy "vlogger" visual markers — floating text overlays, bright graphical elements — compared to Kyrgyz content?
4. **Color composition**: Do Kyrgyz and Russian-language videos differ in their dominant color profiles (hue, red/yellow/blue ratios)?

## Data

### Source
The primary dataset (`data/raw/all_urls_labels.csv`) is a corpus of YouTube video URLs collected during a sock-puppet algorithmic audit of YouTube recommendations in Kyrgyzstan. Each row is a unique video. Videos were labeled by research assistants at a university in Bishkek for:

- **Language spoken**: Russian, Kyrgyz, English, unknown, or other
- **Apparent ethnicity of speakers/characters**: Slavic, Kyrgyz, other Central Asian, Caucasian, other, no people
- Additional flags for unreachable/deleted videos

> **Note:** The raw CSV is not stored in this repository due to size. Contact the authors or see the Zenodo archive [link TBD] to obtain it. Place it at `data/raw/all_urls_labels.csv` before running the pipeline.

### Column reference

| Column | Description |
|---|---|
| `url` | Full YouTube URL |
| `is_russian` | `t` if primary language is Russian |
| `is_kyrgyz_x` | `t` if primary language is Kyrgyz (language label) |
| `is_english` | `t` if primary language is English |
| `is_unknown` | `t` if language could not be determined |
| `is_unreachable_x` | `t` if video was unavailable at time of labeling |
| `is_no_language` | `t` if video contains no speech |
| `other_lang` | Free text for other languages (e.g., `kazakh`); also used for `deleted` |
| `is_slavic` | `t` if speakers/characters appear Slavic |
| `is_kyrgyz_y` | `t` if speakers/characters appear Kyrgyz (ethnicity label) |
| `is_other_central_asian` | `t` if speakers/characters appear from other Central Asian groups |
| `is_caucasian` | `t` if speakers/characters appear Caucasian |
| `is_other` | `t` if speakers/characters are other ethnicity |
| `is_no_people` | `t` if no human/character faces present |
| `is_unreachable_y` | `t` if video was unavailable at time of ethnicity labeling |

### Processed data
After running the pipeline, the following file is created in `data/processed/`:

- `visual_features.csv` — one row per video with all computed visual metrics (saturation, luminance, color ratios, text overlay detection)

## Setup

### Prerequisites
- Python 3.9+
- A stable internet connection (for thumbnail fetching)

### 1. Clone the repository

```bash
git clone https://github.com/sharyali05/kyrgyz-youtube-visual-audit.git
cd kyrgyz-youtube-visual-audit
```

### 2. Create a Python environment

**Option A — conda (recommended):**
```bash
conda env create -f environment/environment.yml
conda activate kyrgyz-audit
```

**Option B — pip:**
```bash
python -m venv venv
source venv/bin/activate        # on Windows: venv\Scripts\activate
pip install -r environment/requirements.txt
```

### 3. Install dependencies

```bash
pip install pandas numpy opencv-python requests tqdm easyocr Pillow
```

## Pipeline: Step-by-Step

### Step 1 — Explore the data

Open and run `notebooks/00_data_exploration.ipynb`.

This notebook gives you an overview of the labeled dataset before running any analysis:
- Distribution of language labels (how many Russian, Kyrgyz, English videos?)
- How many videos are deleted or unreachable?
- Cross-tabulations of language vs. apparent ethnicity labels
- Export of clean URL lists per language group

**No downloads required for this step.**

---

### Step 2 — Extract visual features from thumbnails

Run `scripts/02_analyze_thumbnails.py` to fetch YouTube thumbnails and compute visual features. This script fetches only the thumbnail image for each video — no full video download, no ffmpeg required.

```bash
# Fast run — color and saturation only (~5-10 minutes for 200 videos per group)
python scripts/02_analyze_thumbnails.py \
  --csv data/raw/all_urls_labels.csv \
  --output data/processed/visual_features.csv \
  --max 200

# With text overlay detection via EasyOCR (~30-40 minutes additional)
python scripts/02_analyze_thumbnails.py \
  --csv data/raw/all_urls_labels.csv \
  --output data/processed/visual_features.csv \
  --max 200 \
  --ocr
```

**Arguments:**

| Argument | Default | Description |
|---|---|---|
| `--csv` | required | Path to labeled URL CSV |
| `--output` | `data/processed/visual_features.csv` | Path to save output CSV |
| `--max` | `200` | Max videos per language group |
| `--ocr` | off | Enable EasyOCR text overlay detection |

**Features computed per video:**

| Feature | Description | Method |
|---|---|---|
| `mean_saturation` | Average color saturation | HSV S channel mean |
| `mean_luminance` | Average brightness | HSV V channel mean |
| `saturation_std` | Variability in saturation | Standard deviation |
| `luminance_std` | Variability in brightness | Standard deviation |
| `mean_hue` | Dominant hue | HSV H channel mean |
| `red_ratio` | Proportion of red/orange pixels | HSV hue mask |
| `blue_ratio` | Proportion of blue pixels | HSV hue mask |
| `yellow_ratio` | Proportion of yellow pixels | HSV hue mask |
| `text_overlay` | Whether text is detected in thumbnail | EasyOCR (requires `--ocr`) |
| `n_text_chars` | Number of characters detected | EasyOCR (requires `--ocr`) |
| `thumbnail_found` | Whether thumbnail was successfully fetched | — |

**Notes:**
- Videos that are deleted, private, or unreachable will have `thumbnail_found = 0` and `NaN` for all visual features
- Thumbnails are fetched in order of quality: `maxresdefault` → `hqdefault` → `mqdefault` → `default`
- YouTube returns a small grey placeholder for missing thumbnails — these are automatically detected and skipped

---

### Step 3 — Statistical analysis

Open and run `notebooks/03_statistical_analysis.ipynb`.

This notebook performs the core comparisons between language groups:
- **Descriptive statistics** — means, medians, standard deviations per feature per language group
- **Group comparisons** — Kruskal-Wallis tests comparing Russian vs. Kyrgyz vs. English groups
- **Post-hoc pairwise tests** — Mann-Whitney U tests with Bonferroni correction
- **Effect sizes** — rank-biserial correlation to report practical significance
- **Correlation analysis** — relationships between visual features

Results are saved to `results/tables/`.

---

### Step 4 — Visualization

Open and run `notebooks/04_visualization.ipynb`.

This notebook generates publication-ready figures:
- Boxplots of saturation and luminance by language group
- Bar chart of text overlay frequency by language group
- Color ratio comparison across language groups
- Scatter plots for feature correlations
- Example thumbnail grids for qualitative illustration

Figures are saved to `results/figures/` as `.png` and `.pdf`.

## Connecting Results to Theory

The visual features computed here operationalize claims from ethnographic interviews and the language ideology literature:

| Community claim | Operationalization |
|---|---|
| "Kyrgyz cartoons are static / boring" | Mean saturation, luminance, color variability |
| "Russian content is more colorful / professional" | Mean saturation, mean luminance |
| "Russian content has vlogger aesthetics" | Text overlay frequency, yellow/red ratios |
| Soviet-era aesthetic associations | Visual similarity clustering across language groups (exploratory) |

Findings should be interpreted alongside qualitative interview data, not in isolation. The goal is triangulation: do computational patterns confirm, complicate, or contradict what community members report?

## Limitations

- **Thumbnail as proxy**: Thumbnails are representative but not identical to video content. They are designed by creators for maximum visual appeal, which may amplify rather than accurately reflect the video's overall aesthetic.
- **Video availability**: Some videos have been deleted since original data collection. The `is_unreachable` flags track known deletions, but additional videos may have become unavailable since labeling.
- **Text overlay detection**: EasyOCR detects Latin and Cyrillic script well but may underperform on stylized fonts common in YouTube thumbnails.
- **Animation vs. live action**: Color metrics may behave differently for animated content vs. live-action vlogger content. Consider analyzing these subgroups separately.
- **Language label quality**: Language labels were applied by human annotators. Inter-rater reliability scores should be reported if available.
- **Causal inference**: This study is observational. Visual differences between language groups reflect the content available on the platform; they do not establish that visual aesthetics cause children's language preferences.
